# FakeEcommerce API Tests

BDD tests for JSONPlaceholder fake API endpoints using Reqnroll (SpecFlow), HttpClient, and FluentAssertions.

## Features tested

- GET /posts
- GET /posts/{id}
- POST /posts
- Contract tests for response structure, status codes, and headers

## Run tests

1. Open solution in Visual Studio 2022
2. Build project
3. Open Test Explorer
4. Run all tests
